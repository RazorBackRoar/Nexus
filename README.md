> **Native macOS Safari bookmark manager and batch URL opener.**
> Organize bookmarks, batch open URLs, and manage your browsing workflow with a dark native interface.

---

## Features

- **Safari Integration** ‚ÄĒ batch open URLs in Safari via AppleScript
- **Hierarchical Bookmarks** ‚ÄĒ drag-and-drop folder organization
- **Smart URL Extraction** ‚ÄĒ find valid links in any pasted text
- **Private Browsing** ‚ÄĒ one-click stealth/private mode support
- **Export / Import** ‚ÄĒ back up your collections as JSON
- **Themeable UI** ‚ÄĒ five muted dark color themes with per-tab accents
- **Apple Silicon Native** ‚ÄĒ arm64 build optimized for M-series Macs

---

## Installation

1. Download the latest `Nexus.dmg` from [Releases](https://github.com/RazorBackRoar/Nexus/releases)
2. Open the DMG and drag `Nexus.app` to `/Applications`
3. First launch ‚ÄĒ right-click the app ‚Üí **Open** to bypass Gatekeeper on the ad-hoc signed build
4. Go to **System Settings ‚Üí Privacy & Security ‚Üí Automation** and enable **Safari** for Nexus

---

## Usage

1. **Add Bookmarks** ‚ÄĒ click `+` or paste URLs directly
2. **Organize** ‚ÄĒ create folders and drag to rearrange
3. **Batch Open** ‚ÄĒ select multiple bookmarks ‚Üí **Open in Safari**
4. **Extract URLs** ‚ÄĒ paste any block of text and Nexus finds all valid links automatically

### Themes

Nexus ships five muted dark themes, each with distinct accent colors for the Safari, Bookmarks, and Settings tabs:

| Theme | Notes |
|-------|-------|
| **Midnight Blue** | Default |
| **Rose** | Warm pink accents |
| **Forest** | Green and gold |
| **Violet** | Purple and blue |
| **Ember** | Orange and coral |

Pick a theme under **Settings ‚Üí Theme**. Upgrades from older builds automatically migrate legacy neon theme names (e.g. `Neon Blue` ‚Üí `Midnight Blue`).

## Development

### Requirements

- Python 3.14
- macOS 12.0+
- [uv](https://github.com/astral-sh/uv)

### Setup

```bash
git clone https://github.com/RazorBackRoar/Nexus.git
cd Nexus
# Workspace layout: sibling Apps/.razorcore provides editable razorcore
uv sync
uv run python -m nexus.main
```

### Build

```bash
razorbuild Nexus
# Output: dist/Nexus.dmg
```

### Lint & Test

```bash
uv run ruff check .
uv run ty check src --python-version 3.14
uv run pytest tests/ -q
```

### Developer notes

**File logging is opt-in** because pasted URLs may be sensitive. Console logging is always on.

| Variable | Effect |
|----------|--------|
| `NEXUS_LOG_DIR` | Write logs to this directory (implies file logging) |
| `NEXUS_ENABLE_FILE_LOGGING=1` | Enable file logging at the default Application Support path |

Sensitive values in logs are fingerprinted via `privacy_fingerprint()` in `src/nexus/core/config.py`.
